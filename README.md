# SortMii
This visualizes sorting algorithms on your Wii.

## Algorithms
- Bubble Sort
- Cocktail Shaker Sort
- Gnome Sort
- Insertion Sort

## Controls
- D-Pad Up and Down to increase/decrease array size
- D-Pad Left and Right to switch between algorithms
